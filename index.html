Recent Work



Hybrid Movie Recommender

Recommender engines filter information based on user’s likes or preferences and predict user’s rating or preference on a new item. Given a database of users and movie ratings, this project aims at building a recommender engine that predicts a ranked list of movie recommendations that the user would like to watch. I have combined content-based filtering that uses Natural Language Processing & Sentiment Analysis and user-based collaborative filtering to overcome flaws of individual filtering approach like cold-start, subject-matter expertise dependency and popularity bias. Using the hybrid recommender, I have built a streamlit app where users rate the movies they watched and get personalized movie recommendations.

Disparity in COVID Vaccine across The United States

What does COVID vaccine response look like in the United States at a county level? Does the response reveal stratification within states along similar lines to those seen in the 2020 presidential election? If there is a close relationship, can we predict how a county voted in 2020 elections using vaccine and vaccine hesitancy related data? In this collaborative project, our team of Data Scientists attempted to answer these questions using unsupervised machine learning model to study county level behavior towards vaccine and supervised machine learning model to predict 2020 county election results.

Classification of Reddit Posts

Is it possible to add novice/beginner vs expert level feature next to reading time on Engineering articles? This project aims at identifying keywords used by beginners vs experts in the Engineering field and use these keywords to predict whether a post is fit for a beginner or an expert. By scraping data from Reddit’s Pushshift API, I have built a classification model that differentiates between posts from r/explainlikeimfive (forum for layperson-friendly explanations) and r/AskEngineers (forum for science and technology) using Natural Language Processing and Sentiment Analysis.

Prediction of House Prices in Ames,IA

Are you planning to sell your house? How much will be the sales price? What are some of the features that are adding or bringing down the value of your house? I have attempted to answer these questions by developing an algorithm using Linear, Lasso and Ridge Regression models that predicts house price. The model is optimized for metrics like RMSE and R2 score, and the best model has been submitted on Kaggle.
